# 브랜드 로고
-브랜드명 : VELOR 
-브랜드 의미 : Velocity(미래, 진보, ) + Or(프랑스어로 금)
-브랜드 철학 : ai가 주거민의 생활을 학습해서 좀 더 똑똑하고 더 편안하고 더 가치있는 공간을 만들기 위해 노력함
-타겟 : 일상의 AI 활용에 대한 관심이 많은 20대~40대
-USP : 사람이 ai에 맞추는 것이 아니라 ai가 사람에게 맞춰지는 주거공간
-핵심 메세지 : Where intelligence Lives.
-로고 색상 : 샴페인 골드, 순백색, 그래파이트 블랙
-로고 디지인 : 미니멀, 럭셔리, 기하학, 프리미엄, 정교한, 깔끔한 선. 균형있는 비율, 로고내에 집,빌딩,지붕,건물, AI칩 제외, 대칭적인 조합, 굵지 않은 선, MINIMAL MORGRAM "V'

# 씬 1 (INTRO)
-씬 0초~1.5초
-화면 구성 : 카메라가 주말 오전 11시쯤으로 보이는 아파트 단지를 높은 고도에서 부드럽게 촬영. 고급스럽고 깔끔한 아파트 건축물, 산책로가 보이며 미래형 주거 공간의 분위기를 강조

#씬 2 
-씬 1.5초~3.8초
-화면 구성 : 일반 가정집 주방 내부, 냉장고 앞에 주부 한명으로 보이는 여성이 뒷모습을 보이며 서있다. 냉장고 도어의 투명 AI 디스플레이가 자동으로 켜진다. 냉장고 속 식재료가 깔끔한 인터페이스로 표시되고 유통기한이 임박한 식재료가 자연스럽게 강조된다. 화면에는 '오늘까지 소비권장 : 우유, 삼겹살'  '오늘의 추천 메뉴 : 크림 파스타' '조리 예상시간 : 20분' '크림파스타 레시피를 띄울수 있는 듯한 크림파스타가 강조된 사각형 아이콘 안의 크림파스타 레시피 아이콘' 들이 표시된다. AI인터페이스는 미니멀하고 세련되며 공간 속에 자연스럽게 녹아든다.

#씬 3 
-씬 3.8초~7초
-화면 구성 : 지하 주차장 . 차량이 지정된 구역에 부드럽게 주차된다. 운전자가 차량에서 내리자 주차장 기둥의 스마트 디스플레이가 자동으로 운전자를 인식하여 활성화된다. 화면에는 ' 타이어 교체까지 1,240KM' '엔진 오일 점검 D-18' '베터리 상태 EXCELLENT' 이라는 정보가 심플하고 고급스러운 UI로 표시된다. 운전자는 아무 조작 없이 정보를 확인하며 AI가 일상을 자연스럽게 관리하는 모습을 보여준다.
